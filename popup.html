스크린샷

현재 열린 탭을 캡처해 PNG로 다운로드하거나 클립보드에 복사합니다.

보이는 영역 지금 보이는 부분만 · 전체 페이지 스크롤 포함 한 장 · 영역 선택 페이지에서 드래그해 영역 지정 후 캡처 (다운로드 또는 클립보드)

컬러 피커

페이지에서 클릭한 위치의 색상을 HEX, RGB로 확인합니다.

화면에 보이는 픽셀과 동일한 색을 쓰기 위해, 시작 시 뷰포트를 한 번 캡처한 뒤 그 이미지에서 색을 읽습니다. 「컬러 선택」 클릭 후 잠시 뒤 따라다니는 박스에서 실시간 색상을 확인하고 클릭해 확정하세요. 취소는 Esc.

이미지 최적화

이미지 파일을 리사이즈·압축해 선택한 포맷(JPEG/PNG/WebP)으로 다운로드합니다.

파일 선택 → 포맷·품질 조절 → 「최적화」 클릭 후 저장 위치 선택. PNG는 무손실입니다.

이미지 → Base64

이미지 파일을 선택하면 data URL(Base64) 문자열로 변환해 표시합니다.

이미지 추출

선택한 요소 안의 <img>와 CSS 배경 이미지를 모아 링크로 제공합니다.

「요소 선택」을 누르고 페이지에서 원하는 요소를 클릭하세요. data URL도 그대로 표시됩니다.

선택된 요소 없음

추출된 이미지가 없습니다. 요소를 선택해 주세요.

화면 녹화 (짧은 클립)

이 환경에서는 화면 녹화가 제한되어 있습니다. (조직 정책 등) 스크린샷 기능을 이용해 주세요.

현재 탭 화면과 탭 오디오를 짧게 녹화해 WebM 파일로 저장합니다.

DRM 보호 콘텐츠나 브라우저 내장/특수 페이지에서는 녹화가 차단될 수 있습니다. 팝업을 닫지 말고 녹화 시간을 짧게 설정하세요.

대기 중

URL 인코더/디코더

URL에 쓸 수 있도록 텍스트를 인코딩하거나, 인코딩된 문자열을 다시 읽기 쉬운 텍스트로 디코딩합니다.

Base64 인코더/디코더

텍스트를 Base64 문자열로 인코딩하거나, Base64 문자열을 원문으로 디코딩합니다.

JSON 포맷터

한 줄 JSON을 보기 좋게 정리하거나, 공백을 줄여 미니파이합니다. 검증으로 문법 오류를 확인할 수 있습니다.

텍스트 변환

영문 대·소문자 변환, CamelCase / snake_case / kebab-case 등 네이밍 형식 변환입니다.

해시 생성

입력한 텍스트로 SHA-256 또는 SHA-512 해시 값을 생성합니다.

코드 미니파이어

미니파이: 한 줄로 압축. 뷰티파이: 줄바꿈·들여쓰기로 읽기 쉽게. 코드 붙여넣기 또는 URL 입력 가능.

정규식 테스터

정규식 패턴과 테스트 문자열로 매칭 결과를 확인합니다.

사용법 안내

각 탭의 도구는 아래 순서대로 사용하면 됩니다. 결과가 나오는 도구는 「복사」 버튼으로 클립보드에 복사할 수 있습니다.

미디어

  • 스크린샷 — 캡처할 탭을 연 뒤, 보이는 영역 / 전체 페이지 / 영역 선택 중 하나를 누르면 PNG가 다운로드됩니다. 「영역 선택 (클립보드)」를 사용하면 선택 영역이 클립보드에 복사됩니다. 전체 페이지는 첫 사용 시 로딩이 걸릴 수 있습니다.
  • 컬러 피커 — 「컬러 선택」을 누른 다음, 색을 알고 싶은 웹 페이지로 가서 해당 영역을 클릭합니다. 팝업에 HEX·RGB가 표시됩니다.
  • 이미지 추출 — 「요소 선택」 후 페이지의 요소를 클릭하면, 그 요소와 하위 요소에 포함된 <img>와 CSS 배경 이미지 주소를 모아 보여줍니다. 복사하거나 다운로드할 수 있습니다.
  • 화면 녹화 — 탭 화면과 탭 오디오를 짧게(WebM) 녹화해 저장합니다. DRM 보호 콘텐츠나 특수 페이지에서는 제한될 수 있습니다.
  • 이미지 최적화 — 이미지 파일을 선택하고, 품질(슬라이더)을 정한 뒤 「최적화」를 누르면 JPEG로 저장됩니다.
  • 이미지 Base64 — 이미지 파일을 선택하면 data URL(Base64) 문자열이 표시됩니다. 복사 버튼으로 클립보드에 넣을 수 있습니다.

텍스트

  • URL / Base64 — 입력 칸에 텍스트를 넣고 「인코드」 또는 「디코드」를 누릅니다. 결과 칸 옆 「복사」로 복사할 수 있습니다.
  • JSON — JSON 문자열을 넣고 포맷(들여쓰기), 미니파이(한 줄), 검증(문법 확인) 중 원하는 버튼을 누릅니다.
  • 텍스트 변환 — 영문 텍스트를 넣고 대문자, 소문자, CamelCase, snake_case, kebab-case 중 하나를 선택하면 결과가 나옵니다.
  • 해시 — 알고리즘(SHA-256/SHA-512)을 선택하고 텍스트를 입력한 뒤 「생성」을 누르면 해시 값이 나옵니다.
  • 코드 미니파이어 — CSS/JS 선택 후 코드를 붙여넣거나 스크립트·스타일 URL을 입력하세요. 「미니파이」는 한 줄로 압축, 「뷰티파이」는 줄바꿈·들여쓰기로 읽기 쉽게 만듭니다.
  • 정규식 테스터 — 패턴과 플래그(g/i/m 등), 테스트 문자열을 입력한 뒤 「테스트」를 누르면 매칭 여부·캡처 그룹이 표시됩니다.
  • Compare — 두 텍스트(또는 JSON)를 넣고 「비교」를 누르면 줄 단위로 추가/삭제/동일이 하이라이트됩니다. JSON 모드에서는 포맷 후 비교합니다.

개발

  • 쿠키·스토리지 — 확인할 사이트가 열린 탭을 활성화한 뒤 「쿠키 로드」 또는 저장소 종류 선택 후 「로드」를 누릅니다. 항목별 수정·삭제, 새 키/쿠키 추가가 가능합니다.
  • URL 파라미터 — 「현재 탭 URL 로드」를 누르면 쿼리 스트링이 키·값 카드로 표시됩니다. 카드를 드래그해 순서를 바꾸면 아래에 반영된 URL이 갱신되며, 「복사」로 복사할 수 있습니다.
  • User-Agent — UA 문자열을 넣고 「파싱」을 누르면 브라우저·OS·기기 유형이 요약됩니다. 비워 두면 현재 브라우저 UA가 사용됩니다.
  • 요소 선택기 — 「요소 선택 모드」 후 페이지에서 Inspect 하고 싶은 요소를 클릭하면 태그·크기·스타일 정보가 표시됩니다.
  • 네트워크 모니터 — 「모니터 시작」 후 브라우저 사용 시 발생하는 요청이 목록에 쌓입니다. 「초기화」로 비울 수 있습니다.
  • 성능 메트릭·접근성 검사 — 측정/검사할 페이지 탭을 연 뒤 각각 「측정」「검사 실행」을 누릅니다.

기타

  • 비밀번호 생성 — 길이와 대문자/소문자/숫자/특수문자 옵션을 선택한 뒤 「생성」을 누르면 안전한 랜덤 비밀번호가 나옵니다.
  • Lorem Ipsum — 단어 수 또는 문단 수를 지정하고 「생성」을 누르면 더미 텍스트가 생성됩니다.
  • JWT 디코더 — JWT 문자열을 붙여넣고 「디코드」를 누르면 헤더·페이로드가 JSON으로 표시됩니다. 서명 검증은 하지 않습니다.
  • 단위 변환 — 길이·무게·부피·넓이 카테고리와 단위를 선택한 뒤 값을 입력하면 결과가 표시됩니다.
  • 퍼센트 계산 — 전체값의 비율값·일부값의 %, 증감값·증감률을 입력한 뒤 「계산」을 누르면 결과가 나옵니다.
  • QR 코드 — URL 또는 텍스트를 입력하고 「생성」을 누르면 QR 이미지가 나타납니다.
  • 시간 변환 — 타임스탬프(숫자)를 넣으면 날짜로, 날짜 문자열을 넣으면 타임스탬프로 변환됩니다.
  • 탭 관리 — 「탭 저장」으로 현재 탭 목록 저장, 「탭 복원」으로 저장된 탭 다시 열기, 「중복 닫기」로 같은 URL 탭만 남깁니다.
