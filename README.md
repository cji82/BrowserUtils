# BrowserUtils - 개발자 도구 모음

개발에 유용한 도구들을 하나의 크롬 확장 프로그램으로 모아 둔 팝업형 유틸리티입니다.  
스크린샷, 인코딩, 쿠키·스토리지, 네트워크 모니터 등 자주 쓰는 기능을 확장 아이콘 클릭 한 번으로 사용할 수 있습니다.

---

## 주요 기능

### 미디어
| 기능 | 설명 |
|------|------|
| **스크린샷** | 현재 탭을 캡처. 보이는 영역 / 전체 페이지 / 영역 선택(다운로드 또는 클립보드) |
| **컬러 피커** | 페이지에서 클릭한 위치의 색상을 HEX·RGB로 확인 |
| **이미지 최적화** | 이미지 파일을 리사이즈·압축해 JPEG/PNG/WebP로 다운로드 |
| **이미지 Base64** | 이미지 파일을 선택하면 data URL(Base64) 문자열로 변환 |
| **이미지 추출** | 선택한 요소 안의 `<img>`·CSS 배경 이미지 URL을 모아 표시 (복사·다운로드) |
| **화면 녹화** | 현재 탭 화면·오디오를 짧게 WebM으로 녹화 (DRM/특수 페이지 제한 있을 수 있음) |

### 텍스트
| 기능 | 설명 |
|------|------|
| **URL 인코더** | URL 인코딩/디코딩 |
| **Base64** | 텍스트 Base64 인코딩/디코딩 |
| **JSON** | 포맷(들여쓰기), 미니파이, 검증 |
| **텍스트 변환** | 대·소문자, CamelCase / snake_case / kebab-case |
| **해시** | SHA-256, SHA-512 해시 생성 |
| **코드 미니파이어** | CSS/JS 미니파이·뷰티파이 (붙여넣기 또는 URL 입력) |
| **정규식 테스터** | 패턴·플래그·테스트 문자열로 매칭·교체·상세 결과 확인 |
| **Compare** | 두 텍스트(또는 JSON) 줄 단위 비교, 차이 하이라이트 |

### 개발
| 기능 | 설명 |
|------|------|
| **쿠키** | 현재 탭 도메인 쿠키 조회·수정·삭제·추가 |
| **스토리지** | LocalStorage / SessionStorage 조회·수정·삭제·추가 |
| **URL 파라미터** | 현재 탭 URL 쿼리 스트링을 키·값 카드로 표시. 드래그로 순서 변경, 편집·삭제·추가 후 반영 URL 복사 |
| **요소 선택기** | 페이지에서 클릭한 요소의 태그·크기·스타일 정보 표시 |
| **네트워크 모니터** | 확장 사용 중 발생하는 웹 요청 목록 표시 |
| **성능 메트릭** | 현재 탭 페이지 로드 시간, 리소스 수 등 간단 성능 수치 |
| **접근성 검사** | 이미지 alt, 레이블 등 접근성 이슈 검사 |
| **User-Agent** | UA 문자열 입력 시 브라우저·OS·기기 유형 요약 (비우면 현재 브라우저 UA) |

### 기타
| 기능 | 설명 |
|------|------|
| **비밀번호 생성** | 길이·문자 종류 선택 후 랜덤 비밀번호 생성 |

| **JWT 디코더** | JWT 문자열 디코딩 (헤더·페이로드 JSON 표시) |
| **단위 변환** | 길이·무게·부피·넓이 단위 변환 |
| **퍼센트 계산** | 비율·증감률 등 퍼센트 계산 |
| **QR 코드** | URL/텍스트 입력 후 QR 이미지 생성 |
| **시간 변환** | 타임스탬프 ↔ 날짜 문자열 변환 |
| **탭 관리** | 탭 목록 저장·복원, 중복 URL 탭 정리 |

## 설치 방법

1. **저장소 클론**
   ```bash
   git clone https://github.com/YOUR_USERNAME/BrowserUtils.git
   cd BrowserUtils
   ```

2. **Chrome에서 확장 프로그램 로드**
   - Chrome 주소창에 `chrome://extensions/` 입력 후 접속
   - 우측 상단 **개발자 모드** 켜기
   - **압축해제된 확장 프로그램을 로드합니다** 클릭
   - 클론한 `BrowserUtils` 폴더 선택

3. **사용**
   - 브라우저 툴바의 확장 아이콘을 클릭해 팝업을 띄운 뒤, 원하는 탭(미디어·텍스트·개발·기타)에서 도구를 선택해 사용합니다.

### 요구 사항
- **Chrome** (또는 Chromium 기반 브라우저)  
- Manifest V3 지원 버전

## 프로젝트 구조 (요약)

```
BrowserUtils/
├── manifest.json      # 확장 메타데이터·권한
├── popup.html         # 팝업 UI
├── popup.js           # 팝업 로직·도구 연동
├── popup.css          # 팝업 스타일
├── background.js      # 서비스 워커
├── content.js         # 페이지 주입 스크립트 (요소 선택, 컬러 피커 등)
├── icons/             # 확장 아이콘 (16/32/64/128)
├── libs/              # 서드파티 (html2canvas, qrcode 등)
├── modules/           # 기능별 모듈 (스크린샷, 쿠키, 인코더 등)
│   └── easter_egg/    # 게임 등 추가 기능
├── clip-recorder.html / clip-recorder.js  # 화면 녹화
├── diff-viewer.html   # Compare 크게 보기
└── README.md
```
